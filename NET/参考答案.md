# 参考答案

## DLLImport 属性

| 属性              | 作用                                         |
| ----------------- | -------------------------------------------- |
| dllName           | 指定 dll 的名称                              |
| EntryPoint        | 指定 DLL 入口点的名称或序号                  |
| CharSet           | 指定如何向方法发送字符串参数，并控制名称重整 |
| CallingConvention | 指定调用约定                                 |
